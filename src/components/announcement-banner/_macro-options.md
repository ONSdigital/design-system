| Name        | Type   | Required | Description                                                                                      |
| ----------- | ------ | -------- | ------------------------------------------------------------------------------------------------ |
| title       | string | true     | The title for the announcement banner                                                            |

| link        | int    | true     | The link for the announcement banner                                                             |
